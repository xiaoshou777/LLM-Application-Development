# 带上下文记忆的 OpenAI 聊天机器人
一个简洁、稳定、带对话记忆、异常处理完善的 Python 版 OpenAI 聊天机器人，支持多轮对话、自动修剪上下文、日志打印、记忆清空。

## 功能特点
- ✅ 多轮对话上下文记忆
- ✅ 自动修剪上下文长度（防止 Token 超限）
- ✅ 完善的异常处理（认证、网络、接口、未知错误）
- ✅ 清晰的日志输出（时间 + 级别 + 信息）
- ✅ 支持清空记忆 / 退出程序
- ✅ 环境变量配置，安全不泄露密钥

## 环境要求
- Python 3.8+
- 有效的 OpenAI 兼容 API 服务（官方 / 第三方代理均可）

## 安装依赖

pip install -r requirements.txt
